# Moodle C-Test Generator

<img src="logo.svg" width="100%">

## About

With this tool so-called C-Tests can be created without any effort and automatically downloaded as test questions for Moodle. Alternatively the corresponding test questions can also be processed further in manual mode via the clipboard in Moodle.

The C-Test Generator is browser based, free and licensed unter the MIT license.
